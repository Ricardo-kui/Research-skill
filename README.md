# Research Playbook Skills

This repository packages three local Codex skills extracted from a business-school research vault:

- `research-intro-builder`
- `research-theory-builder`
- `research-revision-preflight`

## What They Do

- `research-intro-builder`
  Rewrites and sharpens scholarly introductions, including opening hooks, first paragraphs, and full introduction structure.
- `research-theory-builder`
  Builds theory logic, mechanism chains, and hypothesis sections.
- `research-revision-preflight`
  Runs preflight checks, desk-review diagnostics, discussion rewrites, and revision support.

## Repository Layout

Each skill folder contains:

- `SKILL.md`
- `agents/openai.yaml`
- `references/standard-input-template.md`

## Intended Use

Copy one or more of these folders into a local Codex skills directory or a workspace `.agents/skills/` directory.

## Notes

These skills were designed for business-school research workflows and journal-facing academic writing.
